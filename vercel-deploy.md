# Orpheus 部署到 Vercel

## 步骤

1. 安装 Vercel CLI
```bash
npm i -g vercel
```

2. 登录
```bash
vercel login
```

3. 部署
```bash
cd C:\Users\vivam\Projects\orpheus
vercel
```

4. 获得 URL
- 自动分配域名：orpheus.vercel.app
- 免费 HTTPS
- 全球 CDN

## 注意
- 音频文件需要放到 public/ 目录 ✅
- 总大小 110 MB 在免费额度内
- 每月 100GB 流量免费
